## Sobre o Projeto
Este projeto consiste na implementação de um processador funcional baseado na arquitetura MIPS, desenvolvido no ambiente Logisim. A arquitetura foi customizada para fins didáticos, operando com um caminho de dados de 8 bits e formato de instrução de 18 bits.

**[Clique aqui para ler o Relatório Técnico e a Arquitetura do Processador](Relatorio_Tecnico_MIPS_GLMV.md)**

## Instruções para Execução
1. Abra o software **Logisim**.
2. Vá em "Arquivo > Abrir", busque a pasta "Processador MIPS" e selecione o arquivo "processador MIPS.circ".
